# NEON VOID

▶ **Play: https://smolsmolstar.github.io/neon-void/**

A neon vertical shooter — upgradeable weapons dropped by enemies, 10 stages of
Dreadnought bosses, a final showdown with DREADNOUGHT OMEGA, endless mode, and a
shared leaderboard. Works on desktop (WASD/arrows + Space · X bomb · P pause · M mute)
and mobile (drag to fly, tap the bomb button).

This repo contains only the deployed site files (GitHub Pages serves `main`).
Development happens in a separate private repo; deploys copy `index.html`,
`game.js`, and `leaderboard.js` here with a bumped `?v=` cache-buster.
